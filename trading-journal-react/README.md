# After Hours' Desk Journal

A professional trading journal application built with React and modern web technologies. Track your trades, analyze performance, and maintain discipline with this comprehensive journaling tool.

## ✨ Features

- **Dashboard Overview**: Real-time P&L tracking, win rate statistics, and performance metrics
- **Trade Logging**: Record detailed trade information with screenshots and analysis
- **Analytics**: Performance breakdowns by session, pair, and time period
- **Calendar View**: Visual trade calendar with daily P&L summaries
- **Trade Gallery**: Screenshot gallery organized by performance
- **Pre-Session Checklist**: Structured preparation protocol
- **Live Exchange Rates**: Real-time USD/PHP conversion
- **Professional Design**: Dark navy, forest green, and crimson color scheme
- **Responsive**: Works on desktop and mobile devices
- **Offline Capable**: Standalone HTML version available

## 🚀 Quick Start

### Option 1: Run Locally (Development)

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Option 2: Standalone HTML Version

Open `trading_journal_v8.html` directly in your browser - no server required!

## 🎨 Design System

- **Primary**: Dark Navy (#002444)
- **Secondary**: Forest Green (#006c46)
- **Error**: Crimson Red (#ba1a1a)
- **Typography**: Work Sans (headings), Inter (body)
- **Spacing**: Consistent 8px grid system
- **Components**: Glass cards, subtle shadows, rounded corners

## 📊 Dashboard Features

- **Live Rate Display**: USD/PHP exchange rate with auto-refresh
- **Performance Metrics**: Total trades, win rate, P&L, best trade
- **Charts**: Cumulative P&L sparkline, win/loss donut chart
- **Session Analysis**: Performance breakdown by trading session
- **Pair Analysis**: P&L by currency pair with visual progress bars

## 🛠️ Tech Stack

- **Frontend**: React 19.2.5
- **Build Tool**: Vite
- **Styling**: CSS Custom Properties (Design System)
- **Icons**: SVG icons
- **Fonts**: Google Fonts (Work Sans, Inter)
- **API**: Exchange rate API for live conversions
