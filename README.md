# After Hours Studio

A polished trading journal project built for the After Hours Desk workflow.

## Overview

This repository contains:

- `trading-journal-react/` — the main React + Vite application
- `trading_journal_v8.html` — standalone version with inlined CSS and JavaScript
- `stitch_after_hours_trading_journal/` — supporting page and component assets for the journal UI

## Features

- Dashboard with real-time metrics
- Trade log and performance analytics
- Trade gallery and calendar review
- Pre-session checklist and journaling workflow
- Responsive design and polished UI

## Quick Start

```bash
cd "$(pwd)"
cd trading-journal-react
npm install
npm run dev
```

## Standalone Version

Open `trading_journal_v8.html` directly in a browser for an offline-capable preview.
