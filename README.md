# After Hours Studio

![License](https://img.shields.io/badge/license-MIT-green.svg)
![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-ready-blue.svg)

A polished trading journal project built for the After Hours Desk workflow.

## Overview

This repository contains:

- `trading-journal-react/` — the main React + Vite application
- `trading_journal_v8.html` — standalone version with inlined CSS and JavaScript
- `stitch_after_hours_trading_journal/` — supporting page and component assets for the journal UI

## Features

- Dashboard with real-time metrics
- Trade log and performance analytics
- Trade gallery and calendar review
- Pre-session checklist and journaling workflow
- Responsive design and polished UI

![Dashboard preview](stitch_after_hours_trading_journal/after_hours_desk_journal_dashboard/screen.png)

## Quick Start

```bash
cd "$(pwd)"
cd trading-journal-react
npm install
npm run dev
```

## Standalone Version

Open `trading_journal_v8.html` directly in a browser for an offline-capable preview.

## GitHub Pages Preview

Enable GitHub Pages in your repository settings using the `main` branch and `root` directory. Once enabled, your app will be available at:

https://antonettegadrinab-cloud.github.io/After-Hours-Studio/

The root `index.html` redirects to the standalone preview file.
